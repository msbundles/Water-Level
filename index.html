Water-Level

This site allows you to input a date and time range and get water level data for Lake Michigan from NOAA.
You can only download a months worth of data at a time. All of the water level data is based on
the International Great Lakes Datum (IGLD), more information on the IGLD can be found here.
All time values are GMT, and all depth measurements are in meters.
More technical information on the API used can be found here.
Once you click "Get Download" a new tab will open a .csv file will be downloaded.
You can use the .csv file in Google Sheets and Excel.
